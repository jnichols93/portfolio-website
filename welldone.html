About the project...


Well-Done is an international non-profit organization, focused on making clean drinking water more available in developing countries. Well-Done partnered with Lambda School, to develop both a User Interface and Database capable of persisting and displaying data collected by their proprietary Mo-Mo Device.

Overview

Well-Done Is an IoT application designed to assist Well-Done in their mission, by persisting, storing, and displaying the data collected by their Mo-Mo sensors. To provide decision-makers, with a tool to see which pumps are down and need maintenance, if the well is running low, and detailed status reports.

Fast Facts

Tech Stack: React, ag-Grid, SASS, Semantic UI, Reactstrap, NodeJS, Postgres, Express, SQL, Knex. Team: Gavin Dreyer, Tim Sharkey, Russ Terry, Luis Martinez, Taylor Hunkler, and myself. My Contributions:
  • Created an activity calendar that uses color to indicate the status by day for an individual sensor and water pump.
  • Used ag-grid to build a grid to display all of the organizations currently using Well-Done's devices in pumps they are maintaining or have installed.
  • Implemented foreign keys on an existing, active database to enhance API functionality.
  • Proactively, fixed bugs and tested code throughout the development cycle.
  • Created a global mixins SASS stylesheet with all colors needed, and common component stylings according to our Figma to make styling changes quick and easy.
